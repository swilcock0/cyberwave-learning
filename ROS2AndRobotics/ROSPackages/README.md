# ROS Packages

This directory contains the Dockerized ROS 2 packages deployed on the hardware.

## Packages

- **[dros2](./dros2/)**: Base ROS 2 Humble environment with Cyclone DDS and introspection tools.
- **[ekf](./ekf/)**: Extended Kalman Filter implementation for sensor fusion (Odom + IMU + Mag).
- **[nav2](./nav2/)**: Configuration for the ROS 2 Navigation 2 stack (Planner, Controller, Recoveries).
- **[rtab](./rtab/)**: Configuration for RTAB-Map SLAM (Simultaneous Localization and Mapping).
